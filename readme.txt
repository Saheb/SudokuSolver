
Sudoku Solver is a program written in Java which solves all sorts of Sudoku puzzles, to find a solution or test whether it is running perfectly or not you can follow the steps below:

a) Run the SudokuSolver.java file using Eclipse or using javac SudokuSolver.java in terminal

b) Type the Sudoku with "0" in place of empty boxes

c) After you type the entire Sudoku, press Enter and you will see the solution.
